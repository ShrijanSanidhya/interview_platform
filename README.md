# CodeSync Interview Platform

A real-time CodeSync Platform using the MERN stack and Socket.io.

## Features
- Real-time code syncing via sockets
- Interviewer and Candidate roles
- Built-in chat channel
- Secure JWT based authentication
- Neomorphic UI based on Stitch 'Silk' design system

## Setup
### Backend
\`\`\`bash
cd server
npm install
npm run dev
\`\`\`

### Frontend
\`\`\`bash
cd client
npm install
npm run dev
\`\`\`
